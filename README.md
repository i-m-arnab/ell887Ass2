# Product Management Web App

This directory structure outlines the main files of the Product Management Web Application:

- `app.py`: This file contains the main Flask application code, including routes and logic for handling requests.
- `requirements.txt`: This file lists all the Python dependencies required for the application to run. You can install these dependencies using `pip install -r requirements.txt`.
- `index.html`: This HTML template represents the homepage of the application, displaying a list of existing products.
- `product.html`: This HTML template allows users to add new products to the system by providing product details.


## Features

- Add new products: Users can add new products by providing their name, description, company name, quantity, and cost.
- View all products: Users can view a list of all products currently stored in the database.
- Delete products: Users can delete products from the database.

## Technologies Used

- Flask: Python web framework used for building the backend of the application.
- Azure SQL Database: Cloud-based relational database service used to store product data.
- Bootstrap: Frontend framework for styling the web pages.
- DataTables: jQuery plugin used for rendering and manipulating tabular data.
